# marek.codes

Personal portfolio website for showcasing my web projects, services, and contact options.

## Features
- Clean, responsive UI (mobile-first tweaks included)
- Project cards + dedicated project detail pages
- Language toggle (Czech / English)
- Contact form powered by **Netlify Forms**
- SEO basics: meta description, Open Graph, Twitter card, canonical
- `robots.txt` + `sitemap.xml`

## Tech Stack
- HTML
- CSS
- JavaScript
- Netlify (deployment + forms)

## Pages
- `index.html` (home)
- `project/*.html` (project detail pages)
- `thanks.html` (form success page)

## Running locally
Just open `index.html` in your browser (or use Live Server in VS Code).

## Deployment
Deployed on Netlify.  
Contact form submissions are handled via Netlify Forms.
